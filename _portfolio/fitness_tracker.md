I designed and built an exercise tracker in Python. It is designed to be deployed on single board computers as low power as a Raspberry Pi. Some of the features include tracking of exercise performance, body weight tracking, program planning, and diet monitoring. It supports using either a database or plain text to store data. It supports multiple user interfaces, including modular web frontend. Deployment can be done through a tradtinonal web server or via Docker container
